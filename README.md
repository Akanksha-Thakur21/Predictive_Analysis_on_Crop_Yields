# Predictive Analysis on Crop Yields Using Indian Agriculture Datasets

## Overview

This project was developed as part of my Data Analytics Internship at Develet. The objective of the project is to analyze Indian agricultural data and build a Machine Learning model capable of predicting crop yields based on various agricultural factors.

The project demonstrates the complete data analytics workflow, including data cleaning, statistical analysis, data visualization, feature engineering, predictive modeling, and model evaluation. Using Python and Machine Learning techniques, the model provides insights into crop production trends and predicts crop yield values for future analysis.

This project helped strengthen my understanding of data preprocessing, exploratory data analysis, visualization techniques, and regression-based predictive modeling.

## Objectives

* Clean and preprocess agricultural data.
* Analyze crop production trends using statistical techniques.
* Visualize crop yield patterns.
* Convert categorical data into machine-readable numerical format.
* Build a Linear Regression model for crop yield prediction.
* Evaluate model performance using the R² Score.
* Save predicted results for future analysis.

## Dataset

The dataset contains information related to Indian agriculture, including:

* State Name
* Crop Type
* Crop Name
* Agricultural Production Data
* Crop Yield (Ton per Hectare)

Dataset File:
`Crop_production.csv`

## Technologies Used

* Python

* VS Code

## Project Workflow

### 1. Data Cleaning

* Removed unnecessary columns.
* Handled missing values using mean imputation.
* Verified dataset quality.

### 2. Statistical Analysis

* Generated descriptive statistics.
* Examined data distribution and summary measures.

### 3. Data Visualization

* Created a bar chart showing average crop yield for different crops.
* Identified crop production patterns visually.

### 4. Feature Engineering

* Converted categorical features into numerical values using Label Encoding.

### 5. Model Building

* Selected features and target variable.
* Split dataset into training and testing sets.
* Trained a Linear Regression model.

### 6. Prediction and Evaluation

* Predicted crop yield values on test data.
* Evaluated model performance using R² Score.
* Stored prediction results in CSV format.

## Machine Learning Model

**Algorithm Used:** Linear Regression

Linear Regression was used to establish relationships between agricultural features and crop yield, enabling prediction of future crop production performance.

## Results

The model successfully predicts crop yields based on available agricultural data.

Output generated:

* Predicted Yield Values
* R² Score Evaluation
* Crop Yield Visualization Graph
* Prediction Results CSV File

Output File:

`predictions.csv`
